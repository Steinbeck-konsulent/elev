# PAU-forløb – Se, gå hen, hjælp, fortæl

Lille webapp til mandagens observations- og vejledningsforløb.

## Filer

- `index.html` – selve siden
- `style.css` – design og mobilvisning
- `app.js` – knapper, lokal lagring og vejlederregistrering

## GitHub Pages

1. Opret et nyt repository.
2. Upload alle tre filer til repositoryets hovedmappe.
3. Gå til **Settings → Pages**.
4. Vælg **Deploy from a branch**.
5. Vælg branchen `main` og mappen `/root`.
6. Gem.

Data gemmes kun lokalt i browserens `localStorage`. Der sendes ikke data til en server.
