# PAU-forløb – Version 1.0

Første udgave, juli 2026.

Udviklet af Steinbeck med AI-understøttelse. Indhold og funktion er fagligt gennemgået af et menneske.

## Filer
- `index.html` – forside og brugsramme
- `elev.html` + `elev-v3.js` – elevdelen
- `vejleder.html` + `vejleder-v3.js` – kodebeskyttet vejlederdel
- `style-v3.css` – fælles design
- `logo.svg` – Steinbeck-logo

## Databeskyttelse og brug
- Appen må kun bruges efter institutionens godkendelse og efter institutionens lokale databeskyttelsesregler.
- Brug ikke fulde navne, CPR-numre, helbredsoplysninger eller andre unødvendige personoplysninger.
- Brug kun barnets forbogstav.
- Vejlederdata krypteres lokalt i browseren.
- Data sendes ikke til en server.
- Vejlederdelen skal låses efter brug.
- Data skal slettes, når de ikke længere er nødvendige.
- Appen erstatter ikke institutionens sikre fagsystemer eller journaliseringskrav.
- Appen er ikke en formel prøve og må ikke alene bruges til at konkludere bestået eller ikke bestået.

## Installation på GitHub Pages
Slet gamle filer i repositoryet. Upload derefter alle filer fra denne pakke til hovedmappen, og aktivér GitHub Pages fra `main` og `/root`.
