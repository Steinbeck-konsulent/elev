# PAU-forløb – Version 1.2

Første udgave, juli 2026. Udviklet med AI-understøttelse og fagligt gennemgået.

## Vigtigt ved opdatering fra version 1.1
Eksisterende testdata slettes ikke automatisk. Version 1.2 forsøger ved første åbning at bevare tidligere version 1.1-registreringer i en separat, krypteret historik. De kan læses og udskrives, men indgår ikke i direkte sammenligning med version 1.2.

## Ændringslog 1.2
- Femte støtteniveau genindsat.
- Otte godkendte observationspunkter.
- Separat overførselsfelt fra tirsdag.
- Status og udskrift viser alle gemte felter.
- »Vis status« viser den senest gemte registrering.
- Ældre testdata bevares separat.
- Fjernelsen af oplæsningsmodulet er dokumenteret.
- Ingen score, procent, samlet vurdering eller konklusion.

## Oplæsning
Oplæsningsmodulet er bevidst fjernet, fordi tekstmængden pr. skærm svarer til almindelige tekstkrav i pædagogisk praksis. Vanskeligheder registreres som en observation af, hvad der skete under de givne betingelser. Det forklarer ikke årsagen og er ikke et diagnostisk fund.

## Databeskyttelse
Brug kun anonymt elev-id og barnets forbogstav. Brug ikke fulde navne, CPR-numre, helbredsoplysninger eller andre unødvendige personoplysninger. Vejlederdata gemmes lokalt og krypteret. Elevdelen gemmer ingen personfølsomme oplysninger.

## GitHub Pages
Slet gamle kodefiler i repositoryet, men slet ikke browserdata på enheden. Upload alle filer fra denne pakke til repositoryets hovedmappe.
